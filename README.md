# network-traffic-monitoring

Necessário TCPDUMP e GNUPLOT

É possível iniciar a gravação de um arquivo `.pcap` e/ou gerar os gráficos de utilização da rede para cada ip encontrado em um `.pcap` existente

Para executar:
```
./executavel.sh
```
